SpaceGAN is a deep learning-powered web application designed to enhance space and astronomy images using Generative Adversarial Networks (GANs). The project is part of a thesis by Huzaifa Mazhar and aims to make powerful image enhancement accessible to the public through an intuitive, lightweight platform.

Whether you're a space enthusiast, a student, or a curious user with low-resolution images, SpaceGAN allows you to give your images a meaningful quality boost with just a few clicks. The tool is optimized to work on limited hardware and continues to evolve based on research and real-world use.

This application is built using Flask for the backend and PyTorch for AI modeling. All development was done using open-source tools to ensure free and fair access.
